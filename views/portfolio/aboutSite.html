About this site and MERN stack

This site was made to showcase my talents. It houses my portfolio, and it by itself is part of that portfolio

Chat

I have made a chatbox that has all the typical chatbox features. You can get the last 15 comments, post a new comment, and every comment has dynamic styling in order to make seeing who said what much easier. It pulls the color from the logged in user session.

User accounts

There is a account system in place. You can register an account, log in to it, log out, and view the account details. As a bonus, all passwords are hashed and salted.

Calculator

One of the biggest features of React is one-way data binding. Unfortunately, I haven't found much use for it in many features. But with this engine displacement calculator, I can show off the data binding feature.

Calculate some cubic inches!

React

ReactJS is used here for all of the dynamic elements, and the few often reused elements, like the Navigation Bar. It's written in JSX, and compiled with Babel. A good example of this is the chat box. It makes calls back and forth, updates automatically, and even has dynamic styling.

Bootstrap

One of the worst issues I faced was I going to make this entire thing look good. I tried custom css, and that ended up looking pretty ugly. So now I just use bootstrap.

Passport and BCrypt

One of the biggest concerns I had was securing accounts. Sure, this site probably won't do much more than help get me hired, but I didn't want plain-text passwords either. So I used passport to handle logins (with the local strategy) and bcrypt to actually encrypt the passwords. Worked beautifully.
